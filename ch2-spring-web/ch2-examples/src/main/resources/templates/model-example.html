Thymeleaf 모델 예제

1. 단순 텍스트 출력

메시지: 메시지 내용

<span th:text="${message}">메시지 내용</span>

2. 날짜 포맷팅

현재 시간: 2023-01-01 12:00:00

<span th:text="${#temporals.format(currentTime, 'yyyy-MM-dd HH:mm:ss')}">2023-01-01 12:00:00</span>

3. 객체 속성 접근

사용자 이름: 홍길동

사용자 이메일: hong@example.com

<span th:text="${user.name}">홍길동</span>
<span th:text="${user.email}">hong@example.com</span>

4. 조건부 표시

관리자 권한이 있습니다.

<div th:if="${user.name == '홍길동'}">
  <p>관리자 권한이 있습니다.</p>
</div>

5. 삼항 연산자

상태: 활성

<span th:text="${user.name == '홍길동'} ? '활성' : '비활성'">활성</span>

6. URL 생성

포스트 목록으로
<a th:href="@{/mvc/posts}" class="btn">포스트 목록으로</a>
